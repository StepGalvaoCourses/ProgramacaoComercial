# Aulas

Este diretório contém o material das aulas teóricas do curso de Programação Comercial.

## Estrutura

Cada aula está organizada em seu próprio diretório, contendo:
- Slides ou apresentações
- Exemplos de código
- Notas de aula
- Materiais complementares

## Como Navegar

As aulas estão numeradas sequencialmente. Comece pela Aula 01 e siga em ordem cronológica para melhor compreensão dos conceitos.
