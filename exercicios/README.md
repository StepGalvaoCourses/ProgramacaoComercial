# Exercícios

Este diretório contém exercícios práticos para fixação dos conceitos aprendidos nas aulas.

## Estrutura

Os exercícios estão organizados por tópico ou aula correspondente. Cada exercício inclui:
- Enunciado do problema
- Exemplos de entrada e saída esperada
- Testes (quando aplicável)
- Solução de referência (em diretório separado)

## Como Usar

1. Leia atentamente o enunciado do exercício
2. Tente resolver o problema por conta própria
3. Execute os testes para validar sua solução
4. Consulte a solução de referência apenas após tentar resolver

## Dicas

- Pratique regularmente
- Não desista ao encontrar dificuldades
- Busque ajuda quando necessário
- Revise os conceitos das aulas quando tiver dúvidas
