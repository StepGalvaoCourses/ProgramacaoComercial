# Projetos

Este diretório contém os projetos práticos desenvolvidos durante o curso.

## Estrutura

Cada projeto é uma aplicação completa que integra vários conceitos aprendidos. Os projetos incluem:
- Descrição detalhada do projeto
- Requisitos funcionais e não-funcionais
- Estrutura de código sugerida
- Critérios de avaliação

## Tipos de Projetos

- **Projetos Individuais**: Para prática pessoal e avaliação
- **Projetos em Grupo**: Para desenvolvimento colaborativo
- **Projeto Final**: Projeto integrador que engloba todos os conceitos do curso

## Avaliação

Cada projeto possui seus próprios critérios de avaliação. Certifique-se de:
- Seguir as boas práticas de programação
- Documentar adequadamente o código
- Testar todas as funcionalidades
- Entregar no prazo estabelecido
